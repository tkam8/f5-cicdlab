Environment Setup
================================================

**Goal**: 
----------------
To set up your AWS environment so you can deploy all the resources needed (Jenkins server, F5 BIG-IP and backend servers). 

**Prerequisites**: 
------------------
- Admin account in AWS
- Existing AWS VPC setup with Internet access
